# User Stories — RecipeApp

1. **Signup** — As a new user, I want to create an account with username, email, and password so that I can save my favorite recipes.
2. **Login** — As a returning user, I want to log in with email and password so that I can access my saved data.
3. **Home feed** — As a logged-in user, I want to browse a list of recipes on the home screen so that I can discover new dishes.
4. **Detail view** — As a user, I want to tap a recipe to see full details (image, category, instructions) so that I can cook it.
5. **Favorites (local storage)** — As a user, I want to save recipes to a favorites list persisted on the device so that I can find them later without internet.
6. **External API** — As a user, I want the app to fetch recipes from TheMealDB API so that I always see fresh content.
7. **Settings menu** — As a user, I want a settings icon in the header so that I can access app preferences.
8. **Settings screen** — As a user, I want to toggle dark mode and notifications so that I can customize the app.
9. **Notifications** — As a user, I want to configure and trigger local notifications so that the app can remind me of new recipes.
